
# Github Actions Flask Deployment

This model provisions an automated CI/CD pipeline using GitHub Actions to deploy an application onto AWS EC2 infrastructure provisioned with Terraform.

## ⚙️ What this project does

- Provisions EC2 instances on AWS
- Builds and deploys application updates automatically
- Runs automated testing during code commits
- Automates infrastructure using Terraform
- Builds and deploys application updates automatically

---

## 🧱 Tech Stack

- AWS (EC2, VPC, Security Groups)
- Terraform (Infrastructure as Code)
- Python (Flask application)
- GitHub Actions (CI/CD Pipeline)


### ☁️ Cloud & Infrastructure
- GitHub-hosted CI/CD pipeline execution
- AWS VPC (custom virtual networking)
- AWS EC2 (Linux-based application hosting)

### 💻 Application Layer
- Python Flask web application
- Code validation through CI workflows

### 🔐 Security
- Secure SSH-based EC2 access configuration
- Secrets management through GitHub Actions
- Network isolation using VPC security groups

### 📊 Observability
- GitHub Actions workflow logs
- Application runtime logs
